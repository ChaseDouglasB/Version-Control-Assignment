What is version control and why is it used?
  Version control is a way to keep track and save all different and alternative versions of files, games, code, etc. 
What is the difference between git and github
  Git is the software that executes the file saving, Github is where git sends your files to and where it is stored.

  Git add: Adds a file to eventually be committed and pushed to your repository

  Git push: sends your file(s) to the cloud/Github
